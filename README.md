# cuda-remirrors
CUDA port of remirrors by [estradying](https://github.com/estradying/ReMIRRORS)

## credits
Code is heavily based on [COMMISSION by andrew](https://www.kaggle.com/datasets/xvoid879/commission-by-andrew/). Only the kernels specific to remirrors were written by me.
The algorithm ReMIRRORS was created by estradying.


## compilation
to compile: ```make```
to run:
```
./main --device 0,1 --threads 1 --output output.txt
```
